## （1）应用简介

Around 是一款局域网聊天浏览器应用。简单来说，你可以通过它，让多个**在同一局域网下**的设备中的任意两个相互通信。你也可以进行文件传输——由于局域网受外界环境影响较小，通常能保持相当快的传输速度（最快<font color='green'>**400MB/s**</font>，再也不用忍受微信的🐌速度了~）

## （2）使用说明

①运行Launcher.exe，**将设备连接到同一局域网下**，然后点击“启动应用”。
（如提示无npm，请先配置npm）
②点击URL（本地/网络）跳转到应用网页，也可以通过QR码访问（方便移动端）。
③端口占用问题：**本应用默认端口3000**，如提示被占用，请点击“结束占用进程”后重新启动应用。
④如依赖目录被意外删除，可点击“安装依赖”重新安装。
⑤URL和QR码每30秒自动刷新一次，以适应网络环境改变。
⑥关闭窗口后，应用最小化到系统托盘，可右键点击“退出”关闭应用。

### ②信息的临时与正式属性

我们知道，上传信息或文件的速度取决于网络情况。而根据事务的<del><font color='purple'>**酸性**</font></del>，我们务必需要等待信息完全上传成功，才返回信息ID，以供后续渲染等操作。**但是，如果每次都要等待信息上传才渲染，用户的即时视觉体验并不好。**因此，特添加临时属性，使得信息发出后马上渲染，待成功上传后再转为正式信息。如遇上传失败，则显示红色惊叹号警告。

### ③启动器的部分原理

1.自动运行npm install，根据package.json配置node_modules目录，并生成package_lock.json，**精确描述项目当前实际安装的是哪些包（依赖），确保环境一致性**
2.然后运行npm start，调用server.js启动服务器开启应用
3.server.js继续调用public中的html、css和js文件，将本地项目广播到局域网
4.由于本机作服务器，故本机IP(v4)即服务器IP。此后，通过访问**http://本机IP:端口号（默认3000）**访问应用主页。对于本机，还可以使用localhost代替本机IP。
